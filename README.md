## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Prerequisite

- create `.env` file based on your environment like,
  - for development server > `.env`
  - for staging server > `.env.stage`
  - for production server > `.env.proudction`
- For more information, refer `example.env` file.
- If you're running it with `docker-compose` you need to change database port to `5431`.
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# staging
$ npm run stage

# production 
$ npm run prod
```

## Building
```bash
npm run build
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

### Database commands

- `npm run prisma migrate dev` - to create a migration.
- `npm run prisma db migrate db push` - to run all migration.

### Swagger Documentation

- Swagger documentation endpoint will be running at <b>`http://host:port/docs` </b>.
